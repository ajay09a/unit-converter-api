# Unit Converter

A simple **Unit Converter** web application built with **Node.js (Express)** backend and **Next.js** frontend.  
It allows you to convert **length, weight, and temperature** units easily.

---

## 🛠️ Technologies Used

- **Backend:** Node.js, Express.js  
- **Frontend:** Next.js  
- **Other:** CORS, dotenv  

---

## ⚡ Features

- Convert **length**, **weight**, and **temperature** units
- Clean UI with Next.js
